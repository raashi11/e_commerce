API Endpoints

1. /create-customer

HTTP method: POST

Payload: 

{
    "phone_number": "1234567890",
    "name": "Your name",
    "city": "Your city"
}

2. /create-order

HTTP method: POST

Payload: 

{
    "item_name": "Item's Name",
    "customer_phone": "1234567890"
}

3. /update-order-status

HTTP method: PATCH

Payload: 

{
    "order_id": 1,
    "new_status": "Delivered"
}

4. /fetch-orders-by-city

HTTP method: GET

Parameter: /fetch-orders-by-city?city=Delhi
